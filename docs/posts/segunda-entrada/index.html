Descubriendo Markdown

Publicado el 19 de octubre de 2025

Después de experimentar con Markdown, he aprendido que es una de las herramientas más simples y útiles para escribir contenido estructurado sin complicaciones.

Ventajas de Markdown

  1. Simplicidad: Permite escribir sin distracciones ni etiquetas HTML.
  2. Compatibilidad: Funciona perfectamente en blogs, GitHub y documentación.
  3. Legibilidad: Incluso sin procesar, el texto se entiende fácilmente.

Lo que más valoro

  • Perfecto para escribir documentación técnica.
  • Ideal para publicar en sitios generados con Hugo.
  • Fácil de mantener y compartir en repositorios.
