Mi primer blog creado con Hugo y GitHub Pages

Mi primer blog con Hugo

Publicado el 17 de octubre de 2025

Este sitio web es mi primer proyecto usando Hugo y GitHub Pages. Fue una experiencia muy enriquecedora para aprender sobre la generación de sitios estáticos.

Pasos que seguí

  1. Instalé Hugo en mi máquina con Ubuntu.
  2. Configuré un nuevo proyecto desde cero.
  3. Seleccioné el tema Ananke para el diseño.
  4. Escribí mis primeros artículos en formato Markdown.
  5. Publiqué el sitio directamente en GitHub Pages.

Desafíos encontrados

Al principio tuve algunos inconvenientes con la conexión local y la compilación, pero tras ajustar la configuración del entorno, el sitio funcionó perfectamente.

Conclusión

Hugo es una herramienta excelente para crear sitios web rápidos, seguros y fáciles de mantener. Gracias a su integración con GitHub Pages, publicar un blog nunca había sido tan sencillo.

Recomiendo probarlo a cualquiera que desee tener su propio blog o portafolio sin depender de bases de datos ni configuraciones complicadas.
