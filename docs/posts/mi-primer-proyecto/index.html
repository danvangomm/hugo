Mi primer blog con Hugo

October 17, 2025

Cómo creé este sitio

En este proyecto decidí crear un blog estático usando Hugo y publicarlo con GitHub Pages.

Pasos que seguí

  1. Instalé Hugo en mi máquina Ubuntu
  2. Configuré un sitio nuevo desde cero
  3. Elegí el tema Ananke para el diseño
  4. Escribí los primeros artículos en Markdown
  5. Publiqué el blog en GitHub Pages

Desafíos encontrados

Al inicio tuve algunos problemas con la conexión local y la compilación, pero tras ajustar la configuración del servidor todo funcionó correctamente.

Conclusión

Hugo es una herramienta excelente para crear sitios web estáticos de manera rápida y sencilla.

Recomiendo Hugo para cualquier persona que quiera empezar un blog sin complicaciones.
